># main문

        main 함수는 c 프로그램의 진입점 함수
        진입점 함수는 프로그램이 처음 시작될 때 호출되는 함수.

        실행파일을 메모리로 로드하고 나서 실행파일 안에 있는 main 함수호출

        main 문이 int 반환형이면 정수값을 리턴한다.-> 프로그램의 종료코드
        0이면 정상종료 아니면 비정상 종료라는 것
